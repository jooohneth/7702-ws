---

### 1. Clone the repository

```bash
git clone https://github.com/jooohneth/7702-privy-viem

cd 7702-privy-viem

```

### 2. Install dependencies

```bash
pnpm install
```

### 3. Run the project

```bash
pnpm dev
```
